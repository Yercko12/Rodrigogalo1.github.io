Servicios Contables

Contabilidad Eficiente está alineada con la tecnología de hoy para ayudar a tú empresa a mantener tú información contable y tributaria de forma ordenada, para un correcto cumplimiento tributario mensual.

  • Declaraciones de impuestos mensuales (F29 y F50)
  • Asesoría Contable permanente
  • Registro de la información contable en nuestro sistema online
  • Revisión y contabilización de las Importaciones

Valor : 1 UF

Servicios de Recursos Humanos

Un correcto cálculo de las remuneraciones y cotizaciones previsionales, para esto contamos con personal experto y sistemas acordes para realizar estas tareas de forma eficiente y oportuna.

  • Elaboración de contratos de trabajo
  • Cálculo y confección de liquidaciones y finiquitos
  • Declaración y carga de archivos en plataforma PreviRed para el pago de cotizaciones legales
  • Cumplimiento normativo impartido con la dirección del trabajo

Valor : 2 UF (5 trabajadores, por cada trabajador adicional 0.25 UF)

Servicios de renta

Asesoramiento para un correcto calculo y proceso de impuestos anual de renta, tenemos sistemas acordes para realizar estas tareas de forma eficiente y oportuna.

  • Preparación de Balance para cumplimiento tributario
  • Presentación de declaraciones juradas anuales
  • Declaración y presentación de impuesto a la renta (F22)
  • Presentación de declaraciones de renta de socios

Valor : 4 UF
